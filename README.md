# Eyetracking

Program name:

 Virtual Coach\Release\TobiiTesting1.exe


Pre-installed:
- E4 stream (wifi)
- TobiiProEyeTrackerManager

Setup:
1) Download and release "Virtual Coach.zip"
2) Double click "...\Virtual Coach\Release\TobiiTesting1.exe"


Notes:
The usb hub can only load one camera.
